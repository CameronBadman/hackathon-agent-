# Hackathon Preferences

## Locations
- Brisbane, Queensland, Australia (in-person)
- Fully Online / remote (open to all locations)

## Technical Criteria
- Must be highly technical — software engineering, ML, AI, systems, hardware
- Exclude: business hackathons, design sprints, marketing competitions, startup pitch events, Game hackathons 

## Other
- Prefer events with clear judging criteria and technical challenges
